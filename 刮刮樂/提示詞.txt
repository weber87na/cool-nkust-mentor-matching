# 任務

請設計一張台灣夜市與廟會風格的刮刮樂活動圖片，用於新生與學長姐配對活動。

## 整體風格

- 台灣彩券刮刮樂風格
- 台灣夜市風格
- 台灣廟會風格
- 喜慶、熱鬧、人潮感
- 紅色、金色、黃色為主色調
- 台灣在地設計風格
- 高質感商業宣傳插畫
- 精緻印刷品風格
- 充滿中獎與抽獎氛圍

## 背景元素

可加入：

- 金元寶
- 招財貓
- 發財金
- 紅包
- 春聯
- 鞭炮
- 煙火
- 燈籠
- 夜市燈泡
- 台灣廟宇裝飾
- 金色祥雲
- 財神爺插畫
- 吉祥圖騰

營造台灣彩券與春節活動的感覺。

## 圖片尺寸

- 1080 × 1920
- 直式海報

## 主標題

學長姐配對刮刮樂

## 副標題

刮開看看你的命定學長姐

## 活動標語

- 保證配對成功
- 新生限定活動
- 學長姐帶你飛
- 開學好運抽起來

## 刮刮樂區域

設計四個大型刮刮樂區域：

### 區域 1

位置：

上方中央

造型：

金元寶

### 區域 2

位置：

右上方

造型：

刮刮樂經典圓角區塊

### 區域 3

位置：

左下方

造型：

福袋

### 區域 4

位置：

右下方

造型：

扇形

## 極重要要求

這四個刮刮樂區域必須：

- 完全透明
- 沒有任何內容
- 沒有任何數字
- 沒有任何姓名
- 沒有任何圖案
- 沒有任何灰色塗層
- 沒有任何銀色塗層
- 沒有任何刮除痕跡

僅保留：

- 金色外框
- 裝飾邊框

透明區域將在網頁中透過：

- HTML
- CSS
- JavaScript Canvas

顯示學長姐姓名。

因此必須是真正的透明挖空區域（Transparent Cutout）。

## UI 用途

這張圖片將作為：

- 網頁活動背景
- 刮刮樂互動遊戲
- 新生迎新活動
- 學長姐配對系統

使用。

## 設計要求

- 中央標題醒目
- 四個刮刮區塊位置明顯
- 留足夠空間顯示姓名
- 版面乾淨
- 不要過度複雜

## 禁止內容

不要出現：

- 真實台灣彩券 Logo
- Taiwan Lottery Logo
- QR Code
- 條碼
- 流水號
- 獎金金額
- 中獎號碼
- 真實刮刮樂規則
- 已刮開效果
- 灰色刮膜
- 銀色刮膜
- 揭露內容
- 學長姐姓名

## 輸出要求

- PNG
- 高解析度
- 四個區域具有 Alpha Transparency
- 適合作為網頁 UI 素材
- 保留透明區域供後續程式動態顯示姓名
